# FastAPI Todo App с Docker

Простое приложение для управления задачами (Todo list) на FastAPI с аутентификацией и PostgreSQL в Docker.

## 🚀 Быстрый старт

### Требования
- Docker и Docker Compose (для запуска контейнеров)
- Git (для клонирования репозитория)

### Установка и запуск

1. **Клонируйте репозиторий** (скопируйте код к себе на компьютер):
   ```bash
   git clone https://github.com/kavalvik/fastapi-todos-compose.git
   cd fastapi-todos-compose
